## exp01 说明

### 目标
验证 Panda3D 能否支撑本项目最核心的交互与氛围：第一人称移动、鼠标视角、碰撞防穿墙、昏暗环境光、跟随相机的手电筒聚光灯。

### 操作
- W/A/S/D：移动
- 鼠标：视角
- Shift：奔跑
- F：手电筒开关
- Esc：退出

### 鼠标捕获（重要）
采用点击捕获模式，避免"光标消失又点不到窗口"的死锁：

1. 窗口打开后光标可见，**用鼠标左键点一下画面**进入视角控制；
2. 进入后光标隐藏，可用鼠标转向、WASD 移动；
3. 按 **Esc 释放鼠标**（光标重新出现），再按一次 Esc 退出。

### 常见问题
- **窗口太小：** 已默认 1280×720，可在文件顶部 `win-size` 调整。
- **光标消失、无法移动：** 旧版本一开始就锁定鼠标导致死锁，现已改为点击捕获模式。左键点击窗口开始，Esc 释放。
- **WASD 没反应（按键被输入法拦截）：** 移动键已改为每帧轮询硬件状态，绕开输入法(IME)。若仍异常，可切换到英文输入法，或使用**方向键**（↑↓←→）代替 WASD。
