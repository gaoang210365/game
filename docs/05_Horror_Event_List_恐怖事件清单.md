# 《回声病房》恐怖事件清单 / Horror Event List

> **文档版本：** 0.1（初稿）
> **前置文档：** 01～04 号文档
> **文档日期：** 2026-07-14
> **文档状态：** 事件清单，供实现与调度参考

---

## 1. 目的

本文档汇总《回声病房》的恐怖事件，按强度分级，并为每个事件标注触发条件、表现方式、所属区域和设计目的，供恐怖事件系统实现与节奏调度使用。

分级依据玩法系统文档第 10 节：

- 一级：环境不安（无实际危险）；
- 二级：视觉异象（不立即受伤）；
- 三级：直接威胁（需玩家反应）；
- 四级：追逐（数量有限）。

---

## 2. 事件编号规范

- 一级：A01、A02……
- 二级：B01、B02……
- 三级：C01、C02……
- 四级：D01、D02……

每个事件包含：编号、名称、区域、触发条件、表现、设计目的、备注。

## 3. 一级事件：环境不安

### A01 闪烁的灯管
- 区域：一层门诊大厅、二层主走廊
- 触发：玩家进入区域后随机间隔
- 表现：灯管闪烁、电流声
- 目的：建立不安基调
- 备注：不与危险绑定，制造假警报

### A02 远处的脚步
- 区域：二层主走廊
- 触发：玩家静止或行走时低概率
- 表现：定向脚步声，无实体
- 目的：训练玩家依赖听觉
- 备注：与怪物脚步音色区分

### A03 转动的门把手
- 区域：二层病房
- 触发：玩家靠近特定门
- 表现：门把手缓慢转动后停止
- 目的：制造门后威胁的错觉

### A04 移动的轮椅
- 区域：一层急诊室
- 触发：玩家移开视线后
- 表现：轮椅位置改变
- 目的：引入“观察改变世界”机制

### A05 广播念错姓名
- 区域：全院
- 触发：进入新循环层
- 表现：广播播报错误楼层或陌生姓名
- 目的：暗示空间与时间错乱

### A06 通风管中的响动
- 区域：地下通道
- 触发：玩家经过
- 表现：金属敲击、拖行声
- 目的：地下区域压迫感

---

## 4. 二级事件：视觉异象

### B01 镜中多出的人
- 区域：公共卫生间、值班室
- 触发：玩家照镜子
- 表现：镜中比现实多一个人影，动作慢半拍
- 目的：心理不安，暗示怪物本质

### B02 走廊尽头的人影
- 区域：二层主走廊
- 触发：玩家看向走廊尽头
- 表现：静止人影，转身后消失
- 目的：强化被注视感

### B03 转身后改变的房间
- 区域：二层病房
- 触发：玩家在房间内转身
- 表现：物品位置、床铺状态改变
- 目的：不确定性

### B04 电梯后的走廊
- 区域：中央枢纽旧电梯
- 触发：玩家呼叫电梯
- 表现：电梯门开后里面仍是走廊
- 目的：空间错乱

### B05 病历里的实时记录
- 区域：护士站、病房
- 触发：玩家阅读特定病历
- 表现：病历中出现玩家刚做过的行为
- 目的：暗示循环与被记录

### B06 关灯显现的门牌
- 区域：二层主走廊 408 门
- 触发：玩家关闭手电筒
- 表现：墙面显现被覆盖的“408”
- 目的：谜题线索 + 异象结合

### B07 逐渐靠近的假人
- 区域：治疗室、储物间
- 触发：玩家未照亮假人时
- 表现：假人在视线外靠近
- 目的：静态恐怖压力

### B08 监控里的人影
- 区域：保安室、太平间监控
- 触发：玩家查看监控画面
- 表现：画面中出现现实不可见的人影或通道
- 目的：叙事线索 + 恐怖

## 5. 三级事件：直接威胁

### C01 门外经过的怪物
- 区域：二层病房、地下走廊
- 触发：玩家在房间内，怪物巡逻经过
- 表现：输液瓶碰撞声由远及近再远去
- 目的：教玩家保持安静与关灯
- 玩家应对：停止移动、关灯

### C02 躲藏点被检查
- 区域：储物间、衣柜
- 触发：玩家躲藏且怪物处于搜索状态
- 表现：怪物靠近躲藏点，短暂停留
- 目的：制造躲藏紧张感
- 玩家应对：保持安静，不过早离开

### C03 被迫关灯
- 区域：地下通道
- 触发：怪物进入同一走廊
- 表现：手电筒吸引怪物，玩家必须关灯
- 目的：手电筒风险决策

### C04 停电
- 区域：太平间
- 触发：玩家以错误顺序打开抽屉
- 表现：全区域停电，怪物开始巡逻
- 目的：谜题失败代价
- 玩家应对：使用应急灯，重找正确顺序

### C05 封闭的出口
- 区域：二层、地下
- 触发：剧情节点
- 表现：原本可用出口被封闭，迫使改道
- 目的：制造受困感

### C06 重演的动作
- 区域：二层
- 触发：进入新循环层
- 表现：玩家上一轮的移动被无形存在重复，伴随声音
- 目的：叙事 + 压力

## 6. 四级事件：追逐

追逐数量有限，各承担不同功能。

### D01 首次追逐：教学
- 区域：二层主走廊 → 病房 → 公共卫生间
- 触发：值夜护士首次正式出现
- 表现：怪物发现玩家并追击
- 目的：教授逃生、躲藏、关门规则
- 决策点：至少两个（分岔走廊、躲藏点）
- 结束：玩家进入安全缓冲区，怪物失去目标

### D02 第二次追逐：环境利用
- 区域：地下手术区
- 触发：玩家取得关键道具后
- 表现：怪物追击，玩家需利用切断电源、制造声源等机关
- 目的：要求玩家运用熟悉地图与环境机制
- 决策点：电源开关、声源装置、多条路线

### D03 最终追逐：终章
- 区域：崩坏的二层 → 404
- 触发：终章推进
- 表现：空间崩坏，怪物紧追，结合最终谜题
- 目的：情绪高潮，结合剧情选择
- 结束：进入 404 号病房，触发结局选择

## 8. 节奏与调度原则

- 系统记录近期事件强度，避免连续高压；
- 每次三、四级事件后安排释放期；
- 假警报（一级）与真实威胁（三级）按比例混合，防止脱敏；
- 同一事件避免在短时间内重复触发；
- 二级异象可较频繁，用于维持底层不安；
- 追逐前必须有预警（声音或视觉）。
